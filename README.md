Por cuestiones de temporización (15 minutos), solo contendrá los siguientes puntos:

### 1. Introducción
  - Origen y objetivos
### 2. Arquitectura en capas

  - Protocolos
  - Encapsulamiento
  - Primitivas
  - Venajas
### 3. Modelo OSI

  - Sistemas terminales intermedios
  - Nivel físico

  - Nivel de red
  - Nivel de transporte
  - Nivel de sesión
  - Nivel de presentación
  - Nivel de aplicación
### 4. Equipos de interconexión
  - Repetidor
  - Bridge
  - Gateway (pasarela)
  - Router
  - Switch
